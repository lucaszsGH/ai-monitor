# AI Watchtower starter

This starter contains demo data only.

## Preview on the same computer

From this folder, start any trusted local static-file server. For example, if Python is already available:

```bash
python3 -m http.server 8765 --bind 127.0.0.1
```

Open `http://127.0.0.1:8765` on the same computer.

To use a phone on the same Wi-Fi, bind only to a trusted local interface and apply host firewall rules. Do not expose the port directly to the public internet. HTTPS or a private network is required for a production PWA and remote access.
