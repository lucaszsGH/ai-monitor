# Example prompts

## Generate a safe first sample

> 把我的 iPhone 横屏做成 Claude 和 Codex 状态副屏。先用假数据生成，不要安装工具，不要读取聊天全文。

## Plan a live read-only connection

> 我已经有一个本机 status.json。请按公开状态合同接入横屏页面，只在同一 Wi-Fi 使用，并告诉我哪些字段不该进入前端。

## Audit an existing dashboard

> 检查这个手机 Agent Dashboard 是否分清额度、Context 和工作上下文，并按 DeepWheel、安全区和隐私门禁给出修复清单。

## Remote access boundary

> 我希望离开办公室也能查看状态。先给私人网络方案和风险，不要直接开放公网或安装软件。
