# 安装包与隐私边界

## 当前可用范围

公开仓库目前提供社区 Skill 与合成 Demo。稳定版 Apple 芯片 DMG 只有通过发布清单后才能附加到 GitHub Releases。只有源码 Tag、没有 DMG 资产时，不应视为 App 已发布。

## 支持基线

- Apple 芯片 Mac（M1 或更新）
- macOS 14 或更新版本
- iPhone X 至 iPhone 17 Pro Max 横屏
- 本机使用 Claude Code、Codex 或双开

## App 读取什么

AI Monitor 读取经过隐私过滤的本机状态和用量摘要，不读取或上传对话正文。不同 Provider 的能力并不完全相同：官方账号值与本机推导值会分别标注，不把推断伪装成官方数据。

## 局域网私密链接

面板使用带随机令牌的局域网私密链接。设备必须处于同一局域网并持有完整链接才能访问。请把它当作私人查看邀请：只有确实希望另一台设备或同网他人查看摘要时才分享。

SET 与安装帮助中的分享动作只分享公开 GitHub 项目页，不会分享私密链接、令牌、局域网地址、真实用量或会话截图。

## 未签名安装包

当前版本尚未经过 Apple 公证。请按 DMG 内的图文安装指南，在“系统设置 → 隐私与安全性”中为 AI Monitor 完成一次放行。无需关闭 macOS 安全保护，也无需运行终端命令。
