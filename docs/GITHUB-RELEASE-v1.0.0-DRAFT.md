# AI Monitor v1.0.0｜把 iPhone 变成 Claude + Codex 副屏

把闲置或充电中的 iPhone 横放在 Mac 旁，持续查看哪些会话正在等你、订阅额度还剩多少、最近主要使用了什么模型，不用反复切换 Claude 与 Codex 窗口。

## 你会得到

- Claude 单开、Codex 单开或双开状态副屏；
- 等待、执行、待验收等高频状态提醒；
- 订阅额度、用量诊断与模型占比；
- iPhone X 至 iPhone 17 Pro Max 横屏适配；
- 本机处理状态，不读取或上传对话正文。
