# Intro assets

Editable SVG and rendered PNG hero/workflow pairs are provided in English and Simplified Chinese.

`watchtower-image2-concept-v2.png` is the refined image2 visual concept used as a design reference, not as executable UI. It establishes strict 1:1 agent parity, restrained semantic color, compact status pills, and a single blue action focus.

`watchtower-phone-landscape-apple-17promax-final.png` is the executable equal-split HTML/CSS implementation rendered at 956×440 logical points and 3× output, producing 2868×1320 physical pixels. It injects a conservative 62px horizontal safe-area guard for rendering QA.

`watchtower-phone-landscape-apple-iphonex-final.png` is the same implementation rendered at 812×375 logical points and 3× output, producing 2436×1125 physical pixels. It injects a conservative 44px horizontal safe-area guard for rendering QA.

Earlier failed and superseded local render iterations were removed after owner approval and must not be recreated in a public commit or release.
