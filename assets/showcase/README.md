# AI Monitor public showcase assets

These 1600×900 PNG files use the approved DeepWheel public visual system and synthetic demo data. Product surfaces are rendered from the formal AI Monitor UI rather than redrawn for marketing.

- `01-dual-now-*`: dual-tool NOW view
- `02-single-adaptive-*`: Claude-only and Codex-only adaptive views
- `03-limits-*`: subscription limits and reset semantics
- `04-usage-models-*`: usage diagnostics and model mix
- `05-set-privacy-*`: language, provider selection, and privacy boundary
- `06-install-*`: three-step Mac-to-iPhone onboarding

No image contains a real LAN address, access token, account identifier, private session, or customer data.
